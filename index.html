Pipoquinha Games

Olá ser tridimensional que estar a ler esse texto, bem vindo ao site da Pipoquinha Games :3

Esse é o nosso primeiro projeto, o qual foi dado a alcunha de "Pipoquinha", nesse jogo você controla um personagem, o qual deve desviar dos obstáculos para não acabar morrendo.

Inicia-se o game com 3 corações, a cada dano tomado,um coração é perdido,mas não te preocupes, para recuperar tais corações é só pegar os antibióticos que irão aparecer ao longo do game : )
